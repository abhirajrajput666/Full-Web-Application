## 🐛 TROUBLESHOOTING

```bash
npm install
```

### "Port already in use"
Edit `.env` and change `PORT=5000`

### "Database locked"
Delete `study_tracker.db` and restart

### "Cannot connect to server"
- Verify server is running
- Check `http://localhost:5000`
- Check browser console (F12)
